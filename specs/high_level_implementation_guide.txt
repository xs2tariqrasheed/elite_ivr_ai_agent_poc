Implement a IVR agent for Elite Limousine using fastapi, mentioned in "call_flow.txt" file.

following are the specs for the implementation:


- Coding guidelines:
    - use python 3.9
    - all code should be clean and simple, easy to understand and maintain.
    - everything seems synchronous, so there's no need to use async/await, but use asyncio where it's absolutely necessary

0- components (fastapi, twilio for realtime call, fasttext for intention recognition, faster_whisper for stt, ollama for llm, logging)
1- Load all relative models (fasttext, faster_whisper, ollama for llm) in the memory before starting the app.
2- Load all audio files in memory before starting the app.
3- Whole call flow is non-interruptible. Non-interruptible means the caller cannot interrupt the agent while the agent is speaking.
4- using python's logging module for logging. Log important event, data and errors.
    e.g. 
        import logging
        logger = logging.getLogger(__name__)
        logger.debug(info)
        logger.exception(info)

5- Make sure the media stream coming from twilio is (mono,16 kHz, float32 PCM, wav) in-memory format before passing to faster_whisper for stt.

6- create call flow state management service.

7- create call flow phases constants and manager.

9- create requirements.txt file for the project.

10- see env.example.txt for all required environment variables, apis keys and other configurations.
